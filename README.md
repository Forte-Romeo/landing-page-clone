# Netflix Landing Page Clone

A responsive Netflix-inspired landing page built using **HTML5** and **CSS3**. This project recreates the look and feel of a modern streaming platform homepage, featuring a hero section, navigation bar, movie showcase area, feature highlights, FAQ section, and footer.

> This project is created for educational and portfolio purposes only and is not affiliated with Netflix.

---

## 📌 Project Overview

The Netflix Landing Page Clone is a front-end web development project designed to practice:

* Semantic HTML structure
* Modern CSS styling
* Responsive web design
* Flexbox layouts
* CSS Grid layouts
* Hero section design
* User interface replication
* Landing page development

The project mimics the design style commonly found on streaming platforms while remaining a standalone educational implementation.


### 2. Open Project Folder

```bash

```

### 3. Launch the Website

Simply open:

```text

```

in your browser.

Or use VS Code Live Server.
